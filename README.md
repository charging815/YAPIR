# Raytracer
A raytracer written in Rust that outputs an image to a PPM file.

## Features
- Rays
- Cameras
- Spheres
- Reflection
- Refraction
- Scattering
- Gamma correction
- Antialiasing
- Depth of field

## Usage
````
cargo init
cargo add rand
cargo run > image.ppm
````
